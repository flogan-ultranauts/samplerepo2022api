FreeCodeCamp APIs and Microservices Projects

Presenting projects as a part of the advanced API curriculum for FreeCodeCamp. I implemented projects using npm packages, Node, APIs, and Databases.

I now qualify for APIs and Microservices certification.

I have built five projects, including:

  • Timestamp Microservice parses timestamp in milliseconds or date string from the API endpoint and returns JSON with timestamp and date.
  • Request Header Parser Microservice gets the client's device's IP address, preferred languages, and system info.
  • URL Shortener Microservice allows posting URLs and receiving a shortened URL that could be copied. Visiting the shortened URL will redirect to the original link.
  • Exercise Tracker allows to create user to add an exercise to any user, to get the user's exercise log optionally for a selected period.
  • File Metadata Microservice allows submitting to a form object that includes a file upload and receiving JSON with file's name, type and size.
